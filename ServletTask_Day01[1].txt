Task 1: Student Information Display

Create a StudentServlet that accepts student details (name, roll, course) from request body.

Configure the servlet in web.xml.

Use doPost() method.

Display student details in plain text format.
Test Your Application with this url:
------------------------------------
http://localhost:8080/StudentApp/student?name=Rahman&roll=101&course=Java

Expected Output:

=== Student Information ===
Name   : Rahman
Roll No: 101
Course : Java



Task 2: Employee Salary Calculator

Create a SalaryServlet that accepts employee details (name, basic, hra, tax) from request body.

Configure the servlet in web.xml.

Use doPost() method.

Calculate net salary = basic + hra – tax.

Display salary breakdown and net salary.

Test Your Application with this url:
http://localhost:8080/EmployeeApp/salary?name=Rahman&basic=50000&hra=10000&tax=8000



=== Employee Salary Details ===
Employee : Rahman
Basic    : 50000.0
HRA      : 10000.0
Tax      : 8000.0
Net Salary: 52000.0

Task 3: Multiplication Table Generator

Create a TableServlet that accepts a number (num) from request body.

Configure the servlet in web.xml.

Use doPost() method.

Generate multiplication table from 1 to 10.

http://localhost:8080/TableApp/table?num=7

num=7
